# Starbucks Coffee Website
A fully responsive Starbucks clone built with Next.js and React, recreating the look and feel of the official Starbucks website with multiple pages, components, and navigation.

# Tech Stack

Framework: Next.js (App Router)
UI: React (JSX components)
Styling: CSS Modules (home.css)
Language: JavaScript / TypeScript (next-env.d.ts)
Database: Neon, Prisma ORM, PostgreSQL

# Getting Started
## Prerequisites

Node.js (v18 or later recommended)
npm or yarn

## Installation
Clone the repository
git clone https://github.com/ngclaire75/StarbucksCoffee
cd mycoffee

Install dependencies
npm install
Running Locally
bashnpm run dev
Open http://localhost:3000 in your browser to view the site.

> Notes

Built using the Next.js App Router structure (app/ directory).
Each page is in its own folder following Next.js conventions.
CSS is handled via a combination of module-level and global styles.
